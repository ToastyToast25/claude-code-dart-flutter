# Error Memory

This directory stores error patterns and their solutions for reference.

## Files

- `build-errors.md` - Compilation and build errors
- `runtime-errors.md` - Runtime exceptions and crashes
- `deployment-errors.md` - Deployment and server errors
- `platform-errors.md` - Platform-specific issues

## Error Entry Format

```markdown
## Error: [Short Description]
**Date**: YYYY-MM-DD
**Frequency**: [count]
**Platforms**: [affected platforms]

### Error Message
```
[exact error message]
```

### Cause
[root cause explanation]

### Solution
```[language]
[fix code or commands]
```

### Prevention
[how to avoid in future]

### Tags
#tag1 #tag2
```

## Usage

Before starting a task, agents should search this directory for related errors to avoid repeating mistakes.
