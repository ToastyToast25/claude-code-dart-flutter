# Pattern Library

This directory stores successful code patterns for reuse.

## Files

- `code-patterns.md` - General Dart/Flutter patterns
- `testing-patterns.md` - Testing strategies and patterns
- `deployment-patterns.md` - Deployment configurations
- `api-patterns.md` - API design patterns

## Pattern Entry Format

```markdown
## Pattern: [Name]
**Category**: code | testing | deployment | api
**Added**: YYYY-MM-DD

### Problem
[what problem does this solve]

### Solution
```dart
[code example]
```

### When to Use
- [situation 1]
- [situation 2]

### When NOT to Use
- [anti-pattern situation]

### Examples in Codebase
- [file:line reference]

### Related Patterns
- [related pattern name]
```

## Usage

Search patterns before implementing features to apply proven solutions.
